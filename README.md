# Customer Churn Analysis and Prediction

## Project Overview
This project analyzes customer churn using SQL Server, Power BI, Python, and Machine Learning. It identifies churn patterns, visualizes key business insights, and predicts customers who are likely to churn using a Random Forest Classifier.

## Objectives
- Analyze customer demographics and service usage.
- Identify factors contributing to customer churn.
- Build interactive Power BI dashboards.
- Predict potential churn customers using Machine Learning.
